# vue-tips

> a tips plugin for vue

## Getting Started
`npm install vue-tips --save`

```
//first import plugin

import VueTips    from 'vue-tips'

//and register the plugin on vue.

Vue.use(VueTips)

//last , you can call it .

this.$tips.show()
```
